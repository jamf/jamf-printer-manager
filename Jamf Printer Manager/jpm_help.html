Jamf Printer Manager Help

Use Jamf Printer Manager to add/remove/modify printers on your Jamf Pro Server

After the app finishes launching you will get a list of locally installed printers (that are not available in Jamf Pro) on the left and a list of printers currently available for deployment on the right.

Adding printers to Jamf Pro:

Printers can be added by either:

  • Double clicking a printer from the list on the left.
  • Selecting one or more printers from the list on the left and clicking the Add button.

Be sure to select a category for the added printer(s) to have that set in Jamf Pro, if desired.

Removing printers from Jamf Pro:

Select one or more printers to be removed from Jamf Pro, then either select the '-' button or hit the delete key on the keyboard. You will be presented with a confirmation dialog to verify the removal:

Note, the local printers available list (left table) does not automatically update as printers are removed from Jamf Pro. If you remove a printer from Jamf Pro and would like it listed again in the list of printers that can be added, click the refresh button:

The printer must be installed locally to reappear in the list.

Modifying a printer in Jamf Pro:

Printer properites can be modified for printers currently available in Jamf Pro. Double click a printer from the list on the right. You will be presented with a new window that allows modifications:

Note, there are 3 different tabs available for modifying the printer properties; General, Definition, and Limitations, similar to what is available in Jamf Pro.

The category for one or more existing printers can be changed by selecting the printer(s) then right-clicking.

To manage printers on another server click the 'Change Server' button located in the top right corner of the mail window. Either select an existing printer or add a new one.

If you find a server listed in the login window that you would like to remove select it while holding down the option key.
